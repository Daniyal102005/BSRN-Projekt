Detailed Description

Diese Klasse stellt die textbasierte Benutzeroberfläche und Netzwerklogik bereit.

Sie verarbeitet Eingaben, lädt und speichert Konfigurationen, und kommuniziert mit dem Netzwerkprozess über Queues.
